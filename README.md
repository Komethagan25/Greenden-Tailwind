# Greenden-Tailwind

  Greenden – Plant Ecommerce Landing Page
  
  Greenden is a responsive plant-themed ecommerce landing page built using HTML and Tailwind CSS.
  The project showcases modern UI design, mobile responsiveness, smooth hover effects, and clean layout structure.

# Live Demo
        live link-https://komethagan25.github.io/Greenden-Tailwind/

# Features
✔️ Fully Responsive Navbar

- Mobile menu icon

- Desktop navigation links

✔️Attractive plant banner image

✔️ About Us Section

- Highlights key brand values

- Icons representing services

✔️ Best Sellers Section

- Grid layout for products

- Hover animations for product images

✔️ Customer Reviews

- Testimonial style cards

- Clean layout and readable content

✔️ Newsletter Signup

- Email input field

- Call-to-action subscribe button

✔️ Footer

- Brand name

- Description

# Technologies Used

- HTML5

- Tailwind CSS 

- SVG Icons
